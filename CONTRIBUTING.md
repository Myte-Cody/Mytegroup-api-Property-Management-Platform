# Contributing

Thanks for helping improve Myte Estate. This repo is a NestJS API.

## Getting started

1. Fork and clone the repo.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure env files:
   - `./.env.example`
4. Run in dev:
   ```bash
   npm run start:dev
   ```

## Development notes

- Use clear commit messages and keep PRs focused.

## Code style

- Format before you commit:
  - `npm run format`
- Tests:
  - `npm run test`

## Reporting issues

Use GitHub issues with steps to reproduce, expected behavior, and environment details.
